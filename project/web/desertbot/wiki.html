Miscellaneous Commands

+help
The +help command can be used to list most of the available DesertBot commands.
+pingme
The +pingme command will ping the user who sends it every 15 seconds, 8 times.
+myinfo
The +myinfo command lists the profile information of the user who sends it. This information includes:
  • The user's discriminator ("#" number, e.g: #0000).
  • The user's nickname and username.
  • The date the user joined Discord.
  • The user's avatar image.
  • A link to the user's profile.
+search
The +search command is similar to the +solve command. The +search command can be used to query the Wolfram Alpha search engine which will return a text-based answer.
+solve
The +solve command is similar to the +search command. The +solve command can be used to query the Wolfram Alpha search engine with a mathematical equation which will return an image-based full explanation of the answer.
+pi
The +pi command will print (roughly) 19,999 digits of the irrational number, Pi, as sourced from the Pi Day Website .

Minecraft Commands

+mcstatus
The +mcstatus command can be used to view the statuses of most of the servers that Minecraft relies on.
+hypixel [username]
The +hypixel command lists the Hypixel Minecraft server profile details of the user whose username is passed as the [username] argument.
+mc [username]
The +mc command lists the profile information of the Minecraft user whose username is passed as the [username] argument. This information includes:
  • Links to download and use the user's skin.
  • The user's Hypixel & HiveMC ranks.
  • The date the user last played on Hypixel or HiveMC.
  • An image of the user's skin.

Please Note: Only currently active Minecraft: Java Edition player names can be passed to these commands. All other usernames - for example, those from Minecraft: Bedrock Edition - will be rejected with an error message.


Fortnite Commands

+fnquests [week number]
The +fnquests command lists all the quests in Fortnite for the week number passed as the [week number]. The week number can be as large as you want, but quests will not display properly for invalid numbers.
+fnmap
The +fnmap command displays the latest map of the Fortnite island.
