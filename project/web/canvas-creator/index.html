Help & Info
  • Text Input | When you use an element that you can add text to, you will need to write your text into the Text Input box.

    To make text Bold, you need to put it in between the bold tags like this: "<b> </b>"
    For example, "These words are normal, <b>these words are bold,</b> these words are normal again!".

    To make text Italic, you need to put it in between the italics tags like this: "<i> </i>"
    For example, "These words are normal, <i>these words are in italics,</i> these words are normal again!".

  • Details | You can use the "Details" element to create a small line of text with an arrow next to it, which can the be clicked on to expand into a larger amount of text (just like this help menu). You will need to set the text summary, and the contents of the element.
  • Usage | Once your document is ready, click "Copy Source Code" and then click the small button in the Canvas text editor that looks like this "</>". If you want to make further edits that aren't available here, press the button again to go to the visual editor.
